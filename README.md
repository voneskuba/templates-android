# Templates

Add to plugins:
```
id("com.github.voneskuba.templates-android") version "0.2.2"
```

Add to settings.gradle.kts:
```
pluginManagement {
    repositories {
        mavenCentral()
        maven { setUrl("https://jitpack.io") }
        gradlePluginPortal()
    }
}
```

Execute gradle task `copyTemplates`
